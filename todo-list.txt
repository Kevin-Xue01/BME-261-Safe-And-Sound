Part 1
1. Apply mask for green color
2. Detect contours
3. Ensure that only large countours pass through filter
4. Approximate polygon on the contours
5. Get the first polygon with side length of 4
6. Find the max pixel and min pixel height of polygon
7. Apply mask for ball color

Part 2
1. Apply mask for ball color
2. Detect contours
3. Ensure that only large countours pass through filter
4. Approximate polygon on the contours
5. Get the first polygon with side length of ~20
6. Find the middle pixel height of polygon
